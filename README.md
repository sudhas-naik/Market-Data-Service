# Market-Data-Service

Production-grade AI-powered stock analysis platform — Market Service microservice.

## Project

The Market Service lives in [`market-service/`](market-service/). See [market-service/README.md](market-service/README.md) for setup, API docs, and architecture.

```bash
cd market-service
cp .env.example .env
docker compose up --build
```
